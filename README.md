# credito-web
FRONT-END DE CONSULTA DE CRÉDITOS ANGULAR 21
- Pré Requisitos:
- Angular 21 e Node v22.22.0

1 - Baixo o projeto e rode a linha de comando para montar o projeto:

   git clone https://github.com/leoaug/credito-web.git	

2 - Depois de baixado execute (com o caminho da pasta do projeto):

	npm install
  

2 - Executar o projeto ele vai ficar em localhost:4200:

	ng serve
