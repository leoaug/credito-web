# credito-web
FRONT-END DE CONSULTA DE CRÉDITOS ANGULAR 21
- Pré Requisitos:
- Angular 21 e com Node v22.22.0

1 - Baixo e o projeto e rode na linha de comando para montar o projeto:

   git clone https://github.com/leoaug/credito-web.git	

2 - Depois de baixado execute:

	npm install
  

2 - Executar o projeto ele vai ficar em localhost:4200:

	ng serve
